## 🎮 The Game

Take control of a ball, collect all the yellow spheres to advance, and survive while dodging relentless enemies. Each level brings new challenges, with 5 progressively harder stages to test your skills.

Get ready to **roll**, **dodge**, and **collect** your way to victory!

## 🚀 How to Run

### Play the build (Windows)
1. Go to the **[Releases page](https://github.com/jenhmy/sphere-run/releases)** 
2. Download `SphereRun.zip`
3. Extract the zip file
4. Run `SphereRun.exe`

> ⚠️ When you try to run the game, Windows might show a warning saying: "Windows protected your PC - Microsoft Defender SmartScreen prevented an unrecognized app from starting". Don't worry! This warning appears because the game doesn't have a paid digital certificate. The game is completely safe. You can check the **[Source Code](https://github.com/jenhmy/sphere-run)**.
>
> **To play anyway:**
> * Click "More info" (not "Don't run")
> * Click "Run anyway"
> * Enjoy the game! 🎮

---

## 🕹️ How to Play

- **WASD keys**: Move the ball. 
- **Goal**: Collect all 🟡 yellow spheres in each level to unlock the exit.
- **Survive**: Don't touch the enemies! Avoid ⚪ white and 🔵 blue balls.
- **Progress**: Complete all 5 levels to win.
